# 🌌 Avatar Quiz

Um quiz interativo desenvolvido com **HTML, CSS e JavaScript** para testar os conhecimentos dos fãs do universo de **Avatar**.

## 📖 Sobre o projeto

O **Avatar Quiz** é uma aplicação web simples e responsiva onde o jogador escolhe um nível de dificuldade e responde perguntas relacionadas aos filmes da franquia *Avatar*.

Ao final do quiz, o jogador recebe sua pontuação, porcentagem de acertos e uma mensagem personalizada baseada no desempenho.

---

## ✨ Funcionalidades

* ✅ Escolha entre três níveis de dificuldade:

  * Fácil (5 perguntas)
  * Médio (10 perguntas)
  * Difícil (15 perguntas)

* 🎲 Embaralhamento automático das perguntas.

* 🖼️ Exibição de imagens relacionadas a cada pergunta.

* 🎯 Feedback visual imediato:

  * Resposta correta destacada em verde.
  * Resposta incorreta destacada em vermelho.

* ⏳ Pequeno intervalo antes da próxima pergunta para melhor experiência do usuário.

* 📊 Tela final com:

  * Quantidade de acertos.
  * Porcentagem de desempenho.
  * Mensagem personalizada.
  * GIF temático baseado na pontuação.

## 🚀 Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/avatar-quiz.git
```

2. Entre na pasta do projeto:

```bash
cd avatar-quiz
```

3. Abra o arquivo `index.html` no navegador.

---

## 🎮 Como jogar

1. Escolha um nível de dificuldade.
2. Clique em **Prosseguir**.
3. Responda às perguntas selecionando uma das opções.
4. Veja imediatamente se acertou ou errou.
5. Confira seu resultado ao final do quiz.
